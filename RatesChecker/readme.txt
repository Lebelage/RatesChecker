	Использованные пакеты:
		1) Newton JSON - для парсинга JSON данных.
		2) OxyPlot - для построения графиков.

	Требования к запуску: 
		.Net Framework 4.7.2
	
	Приложение построено по паттерну MVVM. Основными сервисами программы являются:
		1) EventDispatcherService - отвечает за внутренние ивенты приложения, благодраря которым происходит общение между ViewModels и Services.
		2) NbrbApiService - сервис для общения с API Нац. банка. 
		3) LocalStorageService - сервис, отвечающий за внутренее распределение/хранение/кэширования данных в контексте приложения.
		4) CurrencyPlotService - сервис, отвечающий за построение графика, на основе полученных данных валют.

	Был реализован следующий функционал:
		1) При старте приложения происходит запрос к API для получения состояния соединения и первичных данных(метаданные акутальных валют, для отображения в UI). 
		2) Реализована возможность сохранять данные динамики выбранной валюты в json файл, а так же загружать их оттуда (с возможностью редактирования полей: при изменении таблицы, измененное свойство так же одновляется и в файле).
		3) Построение графика динамики выбранной валюты за выбранный период времени.

	UI был разработан без сторонних инструментов, т.к. Telerik WPF
